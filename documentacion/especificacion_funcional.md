# Especificaciones de las funciones del proyecto 

#### De acuerdo con el alcance definido y los ajustes curriculares, el sistema contará con las siguientes funciones:

1. Gestión de Inventario Animal
	1.1 Registro de Vacas: Captura de datos de raza, peso, fecha de nacimiento, crías y nivel de agresividad
	1.2 Control de Producción Láctea: Registro diario de litros de leche por vaca y estado de salubridad mamaria
	1.3 Registro de Toros: Captura de datos de raza, peso, fecha de nacimiento, nivel de agresividad y estado de castración (capado)
	1.4 Gestión de Propiedad: Registro de propietarios y copropietarios vinculados a cada animal

2. Control Sanitario y Productivo
	2.1 Módulo de Salud: Registro de ganado enfermo y seguimiento de tratamientos
	2.2 Gestión de Medicamentos: Registro y manejo de dosis aplicadas al ganado
	2.3 Control de Alimentación: Registro de dietas y uso de suplementos nutricionales

3. Gestión Administrativa
	3.1 Control de Adquisiciones: Registro de compra y entrada de nuevo ganado a la finca
	3.2 Reportes y Consultas: Visualización ordenada de la información para la toma de decisiones
